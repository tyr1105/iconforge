# 🔨 IconForge — App Icon & Favicon Generator

> Upload one image → get 30+ icons for iOS, Android, Web & PWA. One click. Zero upload. 100% client-side.

**🔗 Live Demo: [tyr1105.github.io/iconforge](https://tyr1105.github.io/iconforge/)**

## Features

- **30+ Icon Sizes** — iOS, Android, Web, PWA, macOS all covered
- **Real-time Preview** — See exactly how your icon looks before downloading
- **Customizable** — Background color, gradient, padding, border radius
- **One-Click ZIP** — Download all icons + manifest.json + HTML tags
- **Zero Upload** — Everything runs in your browser, your images never leave your device
- **Free Forever** — No signup, no limits, no watermarks

## Generated Sizes

| Platform | Sizes |
|----------|-------|
| iOS | 1024, 180, 120, 152, 167, 58, 40 |
| Android | 192, 144, 96, 72, 48, 512 (Play Store) |
| Web/Favicon | 32, 16, 180, 150, 512, 440, 192 |
| PWA | 192, 512, Maskable 192, Maskable 512 |
| macOS | 1024, 512, 256 |

## Usage

1. Open [IconForge](https://tyr1105.github.io/iconforge/)
2. Upload your image (recommended: 1024×1024)
3. Customize background, padding, border radius
4. Click **Download ZIP** to get all icons

## Tech Stack

- Vanilla HTML/CSS/JS
- Canvas API for image processing
- JSZip for ZIP generation
- Zero dependencies, zero server
